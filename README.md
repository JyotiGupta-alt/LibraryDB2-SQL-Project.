# Library Management System (SQL Server)

## Project Description
This project is a Library Management System database created using SQL Server.
It manages books, authors, members, phone numbers, and book borrowing details.

## Database Name
LibraryDB2

## Tables Used
- Authors
- Book
- Member
- MemberPhones
- Borrow

## Features
- Store author and book information
- Track library members
- Maintain multiple phone numbers for members
- Issue and return books
- Identify books not issued
- Track overdue books

## Sample Queries
- Books not issued
- Books currently borrowed
- Total books and members
- Member borrow history

## Technologies Used
- SQL Server
- T-SQL
